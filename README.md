# Adote
## Projeto desenvolvido na Curso da Semana Mult Stack pela treinaweb.com.br

Projeto com o intuito de aprendizado e desenvolvido nas seguintes linguagens:
    
    - Front-End:
      - HTML
      - ReactJS
      - NextJS
      - TypeScript
      - JavaScript
    - Back-End
      - PHP
      - Laravel
